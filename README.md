# Insurance Management System

University project for **Lab Course 1 (Programim), Academic Year 2025/2026**.

The application will manage insurance clients, policies, claims, payments, and insurance types through a secure REST API and a React dashboard.

## Technology Stack

- Backend: Node.js, Express.js
- Frontend: ReactJS
- Styling: Bootstrap
- Database: MySQL
- Authentication: JWT access tokens and refresh tokens

## Project Structure

```text
backend/   Express REST API
frontend/  React application
docs/      Project notes, database design, Trello plan, API documentation
```

## Planned Features

- User authentication and authorization
- Dashboard with insurance statistics
- CRUD operations for clients
- CRUD operations for insurance policies
- CRUD operations for claims
- CRUD operations for payments
- CRUD operations for insurance types
- API documentation and setup guide
